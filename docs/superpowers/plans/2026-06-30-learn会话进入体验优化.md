# /learn 会话进入体验优化 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 优化从 `/learn` 选卡片到 `/learn/:id` 的进入过渡——输入框真位移下沉、消除空态废话闪烁、首条消息格式化为「请教我学习《》」、输入框多行文字不挡 icon。

**Tech Stack:** React 19 + react-router v7 + View Transitions API + Tailwind CSS 4 + Playwright E2E

**Spec:** `docs/superpowers/specs/2026-06-30-learn会话进入体验优化-design.md`

- [ ] **Step 2: 重写 sendMessage，用 VT API 包裹 navigate + 首条消息格式化（单元 C 同时落地）**

当前 `learn.tsx:77-94` 的 `sendMessage`：

```tsx
async function sendMessage(text: string) {
  const trimmed = text.trim();

  try {
```

替换为（含单元 C 格式化 + 单元 A VT 包裹）：

```tsx
// 统一包装为「请教我学习《》」；已含书名号或已是学习请求句式则不重复包装
function formatFirstMessage(text: string): string {
  if (/《.+》/.test(text) || /^(请教|请教我|学习|我想学)/.test(text)) {
    return text;
  }
  return `请教我学习《${text}》`;
}

async function sendMessage(text: string) {
  const trimmed = text.trim();
  if (!trimmed || creating) return;
  setCreating(true);
  const formatted = formatFirstMessage(trimmed);
  try {
    const newSession = await createSession(USER_ID, "未命名对话", teachingMode);
    setSessions((prev) => [newSession, ...prev]);
    const target = `/learn/${newSession.id}`;
    const navigateOpts = {
      state: { firstMessage: formatted, teachingMode },
      replace: true,
    };
    // View Transitions API：同 view-transition-name 元素（ChatInput）跨路由自动位移插值
    // 旧浏览器降级为直接 navigate（无动画但功能正常）
    if (typeof document !== "undefined" && document.startViewTransition) {
      const t = document.startViewTransition(() =>
        navigate(target, navigateOpts),
      );
      t.finished.finally(() => setCreating(false));
    } else {
      navigate(target, navigateOpts);
      setCreating(false);
    }
  } catch (err) {
    console.error("Failed to create session:", err);
    setCreating(false);
  }
}
```
