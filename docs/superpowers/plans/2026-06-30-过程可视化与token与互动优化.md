# Loop 过程可视化 + Token 展示 + 互动优化 Implementation Plan（#1 + #5 + #6体验）

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** ① 每条 AI 消息下加默认折叠的「思考过程」面板，展示 reasoning/工具调用/每步耗时/降级轨迹；② 聊天框底部加双向圆 token 仪表盘（provider 无关，真实用量+估算+会话累计+上下文占比）；③ 互动练习 explore 组件库加齐 5 种（choice/matching/ordering/fill-blank/chart-slider）。

**Architecture:** 消息级面板复用已存未渲染的 tool-call/tool-result annotation + 新增 step-start/step-end/reasoning-delta/failover 事件；token 仪表盘统一读 `LanguageModelUsage`（onFinish 回调），system/tools 用 `estimateTokens` 估算；explore 用 zod discriminatedUnion 扩展，每组件一个 React 文件复用统一 Card 风格。

**Tech Stack:** React 19 + SVG + ai SDK（onFinish/usage）+ Zod + Tailwind CSS 4 + Playwright

**Spec:** `docs/superpowers/specs/2026-06-30-agent-loop体验与健壮性-design.md` 第 3 组

**依赖：** 第 2 组的 SSE 协议统一（§2.1）应已完成（本计划消费 step-start/end/reasoning-delta/failover/usage/context-info 事件）

```tsx
import { useState } from "react";
import {
  ChevronDown,
  ChevronRight,
  Brain,
  Wrench,
  AlertTriangle,
} from "lucide-react";

interface LoopTrace {
  steps: Array<{
    step: number;
    total: number;
    t0: number;
    durationMs?: number;
    reasoning?: string;
    tools?: Array<{ name: string; durationMs?: number }>;
  }>;
  failovers?: Array<{ from: string; to: string; reason: string; step: number }>;
  loopWarnings?: Array<{ type: string; toolName: string; step: number }>;
}

interface LoopTracePanelProps {
  trace: LoopTrace;
}

export function LoopTracePanel({ trace }: LoopTracePanelProps) {
  const [expanded, setExpanded] = useState(false);
  const totalMs = trace.steps.reduce((sum, s) => sum + (s.durationMs ?? 0), 0);

  return (
    <div className="rounded-lg border border-border bg-secondary/20">
      <button
        onClick={() => setExpanded(!expanded)}

        </span>
      </button>
```

```tsx
import { useState } from "react";

interface ChoiceProps {
  label: string;
  options: Array<{ id: string; text: string }>;
  allowMultiple: boolean;
}

export function ChoiceExplore({ label, options, allowMultiple }: ChoiceProps) {
  const [selected, setSelected] = useState<Set<string>>(new Set());
  function toggle(id: string) {
    setSelected((prev) => {
      if (allowMultiple) {
        const next = new Set(prev);
        next.has(id) ? next.delete(id) : next.add(id);
        return next;
      }
      return new Set([id]);
    });
  }
  return (
    <div className="space-y-2">
      <div className="text-xs font-medium text-muted-foreground">{label}</div>
      <div className="flex flex-wrap gap-2">
        {options.map((o) => (
          <button
            key={o.id}
            onClick={() => toggle(o.id)}
            className={`rounded-lg border px-3 py-1.5 text-sm ${
              selected.has(o.id)
                ? "border-primary bg-primary/10 text-primary"
                : "border-border bg-card"
            }`}
          >
            {o.text}
          </button>
        ))}
      </div>
    </div>
  );
}
```

```tsx
  if (item.kind === "input") return <InputExplore key={i} {...item} value={inputValues[i]} onChange={...} />;
  if (item.kind === "choice") return <ChoiceExplore key={i} {...item} />;
  if (item.kind === "matching") return <MatchingExplore key={i} {...item} />;
  if (item.kind === "ordering") return <OrderingExplore key={i} {...item} />;
  if (item.kind === "fill-blank") return <FillBlankExplore key={i} {...item} />;
  if (item.kind === "chart-slider") return <ChartSliderExplore key={i} {...item} />;
  return null;
})}
```

同时清理 iframe 残留：`:20` 的 `source: "iframe" | "manual"` 改为 `source: "manual"`（删除 iframe 字面量）。
