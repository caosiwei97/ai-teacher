#### 2026-05-12 16:30 — Bug 修复：学完小主题后不自动进入下一个

**状态**：✅ 已完成

**事件**：
- 用户反馈：学完一个小主题出现总结打分后，Agent 不会自动过渡到下一个知识点
- 根因分析：`generateAssessmentTool` 和 `advanceNodeTool` 的 `promptSnippet` 仍注入系统 prompt，与 `assessMastery.instruction` 冲突
  - `generateAssessment` 的 promptSnippet："当掌握度 ≥ 80% 时调用"
  - `advanceNode` 的 promptSnippet："当掌握度 ≥ 80% 且已生成评估卡片后调用"
  - 但 assessMastery 的 instruction 要求用 renderUI 替代
  - 模型看到矛盾指令，依次调用 assessMastery → generateAssessment → advanceNode → renderUI，耗尽 stepCountIs(5) 的 step 预算
- 修复：移除冗余工具 + 增加 step 预算

**代码变更**：
- `apps/worker/src/agent/tools/create-tools.ts` — 从 tutorToolDefinitions 移除 `generateAssessmentTool` 和 `advanceNodeTool`（assessMastery 已内部处理节点推进，renderUI mastery-report 已替代 generateAssessment）
- `apps/worker/src/graphs/tutor-graph.ts` — `stepCountIs(5)` → `stepCountIs(7)` 增加安全余量

**验证结果**：
- `pnpm build` 全量通过 ✅

#### 2026-05-12 18:00 — Judge0 → OpenSandbox 迁移（代码执行沙箱）

**状态**：✅ 已完成

**事件**：
- Judge0 在 macOS ARM64 和 WSL2 上无法运行（依赖 cgroups v1）
- 调研并选定阿里开源的 OpenSandbox 作为替代方案（跨平台、Docker 隔离、多语言支持）
- 完成 Docker 镜像拉取（server/execd/code-interpreter，通过 docker.1ms.run 镜像）
- 重写 `sandbox.ts`：从 Judge0 REST API 改为 OpenSandbox 生命周期 API + execd 代码执行 API
- 发现并修复 3 个 API 兼容性问题：
  1. OpenSandbox 创建沙箱需要 `resourceLimits` 字段
  2. OpenSandbox 创建沙箱需要显式 `entrypoint` 参数
  3. OpenSandbox 返回的 endpoint 使用 `host.docker.internal`（Docker 内部 DNS），需转换为 `localhost`
  4. execd SSE 响应是纯 JSON 行（无 `data: ` 前缀），需调整解析逻辑

**代码变更**：
- `docker-compose.yml` — 移除 Judge0 全部 4 个服务（server/worker/postgres/redis），替换为单个 `opensandbox-server` 服务
- `opensandbox.toml` — **新建** — OpenSandbox 服务配置（Docker runtime + 安全策略）
- `apps/server/src/services/sandbox.ts` — **重写** — OpenSandbox API 客户端（长生命周期沙箱 + cached execd URL + SSE 解析）
- `.env.example` — `JUDGE0_URL` → `OPENSANDBOX_URL=http://localhost:2358`
- `.env` — 新增 `OPENSANDBOX_URL=http://localhost:2358`
- `README.md` — 端口表新增 OpenSandbox 2358
- 删除 `judge0.conf`

**验证结果**：
- `curl POST /api/sandbox/execute` Python `print(1+1)` → `{"stdout":"2\n","status":"Accepted"}` ✅
- `curl POST /api/sandbox/execute` JavaScript `console.log(2**10)` → `{"stdout":"1024\n","status":"Accepted"}` ✅
- `curl POST /api/sandbox/execute` Python `import math` → `{"stdout":"π = 3.1416\ne = 2.7183\n","status":"Accepted"}` ✅
- `lsp_diagnostics` sandbox.ts 0 错误 ✅
