# 009 — 学习资料上传 + RAG

> 状态：⬜ 待开始 | 分类：🔵 功能 | 优先级：P3 | 依赖：002

## 目标

用户上传资料，Agent 基于资料教学

## 范围

### Phase 1：文件 + 链接（文本类）

- [ ] PDF 解析（pdf-parse，复杂排版后续升级 Docling）
- [ ] Markdown 解析
- [ ] 链接读取（Jina Reader 优先，fallback Readability + Turndown）
- [ ] 文档分块（Chunk）
- [ ] Embedding（智谱 embedding-3 或同类中文模型）
- [ ] 向量化存储 PostgreSQL (pgvector)
- [ ] 相似度检索 service
- [ ] 上传 UI（文件 + URL 输入）
- [ ] MinIO 文件存储
- [ ] Agent 新增 retrieve-context 工具
- [ ] 验证：上传 PDF / 输入 URL → 系统解析 → 基于资料内容教学

### Phase 2：音视频理解（待讨论）

- [ ] 短内容（<15min）：多模态模型直接理解（Gemini 2.5 Flash）
- [ ] 长内容（>30min）：转录 → 分块 → RAG（成本更优）
- [ ] 视频关键帧截取 + 多模态描述（图表/幻灯片场景）

> Phase 2 方案待实施时再细化，需讨论：模型选型、成本阈值、短/长内容切分策略。
