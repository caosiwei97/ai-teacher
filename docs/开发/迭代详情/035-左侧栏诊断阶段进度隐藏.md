# 迭代 035：左侧栏诊断阶段进度隐藏

> 优先级：P0 | 依赖：034 | 分类：功能
> 覆盖用户反馈：#4（摸底阶段不应展示进度 0/0）

---

## 1. 目标

在诊断摸底阶段（学习路线尚未生成），左侧栏 session 条目不展示"0/0"进度条和进度文字，改为展示"诊断中"状态标签。

---

## 2. 技术方案

```mermaid
flowchart TD
    A[session 数据] --> B{totalNodes > 0?}
    B -->|是| C[显示进度条 + masteredNodes/totalNodes]
    B -->|否| D{status === 'diagnosing'?}
    D -->|是| E[显示"诊断中…"标签]
    D -->|否| F[不显示进度区域]
```

**改动范围极小**：仅 `left-sidebar.tsx` 一个文件。

**当前代码**（第 109-136 行）：
```tsx
const progress = s.progress.totalNodes > 0
  ? Math.round((s.progress.masteredNodes / s.progress.totalNodes) * 100)
  : 0;
// 然后无条件渲染进度条和 {masteredNodes}/{totalNodes}
```

**修改为**：
```tsx
const hasNodes = s.progress.totalNodes > 0;
const progress = hasNodes ? Math.round(...) : 0;
// 条件渲染
{hasNodes ? (
  // 进度条 + 数字
) : s.status === 'diagnosing' ? (
  <span className="text-[10px] text-sidebar-muted">诊断中…</span>
) : null}
```
