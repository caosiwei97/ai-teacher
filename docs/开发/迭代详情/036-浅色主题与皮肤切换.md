## 2. 技术方案

### 2.1 CSS 变量双主题体系

```mermaid
graph LR
    subgraph globals.css
        A["@theme (深色 - 默认)"] --> C[CSS 变量]
        B["[data-theme='light']"] --> D[CSS 变量覆盖]
    end
    
    subgraph 运行时
        E[ThemeProvider] --> F{用户偏好}
        F -->|dark| G["document.documentElement<br/>data-theme='dark'"]
        F -->|light| H["document.documentElement<br/>data-theme='light'"]
        F -->|system| I["matchMedia<br/>prefers-color-scheme"]
    end
```

**CSS 架构**：

当前 `globals.css` 的所有 CSS 变量定义在 `@theme` 块中，作为深色主题默认值。新增浅色主题通过 `[data-theme="light"]` 选择器覆盖同名变量。

```css
/* 深色主题 - 当前默认 */
@theme {
  --color-background: #0f1117;
  --color-foreground: #e8eaed;
  /* ... 70+ 变量 */
}

/* 浅色主题 - 新增 */
[data-theme="light"] {
  --color-background: #fafafa;
  --color-foreground: #1a1a1a;
  --color-primary: #6366f1;
  /* ... 覆盖所有颜色变量 */
}
```

### 2.2 浅色主题配色方案

参考竞品截图 + 护眼研究，浅色主题采用：

| 区域 | 颜色 | 说明 |
|------|------|------|
| 背景 | `#fafafa` / `#ffffff` | 主背景 / 卡片背景 |
| 文字 | `#1a1a1a` / `#6b7280` | 主文字 / 次要文字 |
| 主色 | `#6366f1`（靛蓝） | 保持与深色一致 |

| 代码块 | `#f3f4f6` | 浅灰代码背景 |

### 2.3 ThemeProvider 实现

```mermaid
sequenceDiagram
    participant HTML as document
    participant Provider as ThemeProvider
    participant Storage as localStorage
    participant CSS as CSS Variables
    
    Note over HTML: 页面加载
    HTML->>HTML: 内联 script 读取 cookie/localStorage<br/>立即设置 data-theme（防闪烁）
    HTML->>Provider: React 挂载
    Provider->>Storage: 读取 theme 偏好
    Provider->>HTML: 设置 data-theme 属性
    Provider->>CSS: 变量自动切换
    
    Note over Provider: 用户切换主题
    Provider->>HTML: 更新 data-theme
    Provider->>Storage: 持久化偏好
    Provider->>CSS: 变量自动切换
```

### 2.4 防闪烁方案

在 `<head>` 中注入内联 script，在 React hydration 之前读取 localStorage 并设置 `data-theme`：

```html
<script>
  (function() {
    var theme = localStorage.getItem('theme') || 'dark';
    if (theme === 'system') {
      theme = window.matchMedia('(prefers-color-scheme: light)').matches ? 'light' : 'dark';
    }
    document.documentElement.dataset.theme = theme;
  })();
</script>
```
