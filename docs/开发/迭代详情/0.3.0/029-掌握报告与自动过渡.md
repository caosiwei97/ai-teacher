# 029 — 掌握报告与自动节点过渡

> 状态：✅ 已完成 | 分类：🔵 功能 | 优先级：P1 | 依赖：028
> 完成日期：2026-05-10

## 目标

节点掌握后渲染结构化总结报告（表格 + 徽章），Agent 自动庆祝 + 桥接 + 开始下一节点教学，无需用户手动点击"下一节"。

## 背景

当前掌握通过后，`generateAssessment` 渲染评估卡片，需要用户手动点击"下一节"按钮。缺少掌握的仪式感，过渡不自然。改为自动渲染总结报告 + 无缝过渡到下一节点。

## Checklist

### Phase 1：新增 UI Block 类型

- [ ] `packages/shared/src/schemas/ui-block.ts` 新增 3 种 Schema：
  - `HeadingBlockSchema`: `{ type: "heading", level: 2|3, text: string }`
  - `BadgeBlockSchema`: `{ type: "badge", items: Array<{ text: string, variant: "success"|"warning"|"info" }> }`
  - `MasteryReportBlockSchema`: `{ type: "mastery-report", nodeId, nodeName, score, summary, table: { columns, rows }, badges: string[] }`
- [ ] 注册到 `UIBlockSchema` discriminated union

### Phase 2：前端渲染组件

- [ ] 新建 `apps/web/src/components/ui-blocks/heading-block.tsx` — 标题渲染
- [ ] 新建 `apps/web/src/components/ui-blocks/badge-block.tsx` — 徽章/标签渲染（pill 样式）
- [ ] 新建 `apps/web/src/components/ui-blocks/mastery-report-block.tsx` — 掌握总结报告
  - 标题："第X阶段：{节点名} 总结报告"
  - 表格：核心议题 × 掌握情况
  - 徽章：关键要点（绿色 success 变体）
- [ ] 注册 3 个新 block 到 `apps/web/src/components/ui-blocks/registry.tsx`

### Phase 3：Agent 掌握流程改造

- [ ] 修改 `apps/worker/src/agent/tools/assess-mastery.ts`
  - 得分 ≥ 80 时返回 `instruction` 字段，指导 Agent 自动过渡：
    ```typescript
    instruction: `Node "${nextNodeName}" is now active. Write a brief celebration for the mastered node, then a bridge sentence, then start teaching the new node with a Socratic opening question.`
    ```
  - 返回 `progress` 对象：`{ nodeId, newStatus: "mastered", masteryScore, activatedNextNodeId }`

- [ ] 修改 `apps/worker/src/agent/tools/generate-assessment.ts`
  - 接受总结报告数据：table columns/rows + badges
  - 返回 `renderUI` 兼容的 blocks 数组（heading + table + badge）

- [ ] 修改 `apps/worker/src/agent/prompts/tutor.ts`
  - 新增"掌握后自动过渡"指令模板：
    1. 庆祝掌握（1 句肯定）
    2. 调用 `renderUI` 渲染总结报告（表格 + 徽章）
    3. 桥接句（连接当前节点到下一节点）
    4. 自动开始下一节点的苏格拉底式教学（出第一个引导问题）
  - 指示 Agent 不要等待用户操作，直接开始教学

### Phase 4：移除旧评估卡片的手动过渡

- [ ] 修改 `apps/web/src/components/chat/assessment-card.tsx`
  - 移除"下一节: {nextNodeTitle}"按钮（改为自动过渡）
  - 保留卡片显示作为过渡前的总结展示
