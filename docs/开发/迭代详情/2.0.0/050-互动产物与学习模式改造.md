## 目标

把"对话主导、卡片点缀"翻转为"互动教学产物主导、对话辅助"。本迭代交付 HTML 沙箱渲染器 + 形态 A 互动课 + 教练式 prompt 改造 + 删掌握仪式，兑现核心量化目标：单知识点对话轮次 4-6 → 1-2。

### E2E 策略说明

spec §9.2 Phase 1 要求"E2E 断言掌握后回复字符数/不含固定仪式文本"。但删仪式的效果是 LLM 根据 instruction 生成的文本，E2E 需真实 LLM 多轮对话触发 assessMastery 掌握，慢且不稳定（本项目 E2E 对 LLM 行为断言弱，chat.spec 仅断言"收到回复"）。故：

- 单测层稳定覆盖 instruction 文本变更（TDD）
- E2E 层只确保现有用例不破坏
- 删仪式实际效果纳入人工验收（spec §9.3 已列"对话啰嗦度"为人工指标）
