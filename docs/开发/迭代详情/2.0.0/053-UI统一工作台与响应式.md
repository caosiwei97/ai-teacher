## 子任务① — 顶部 Tab + 渐进解锁 + 三模式主色

### 设计（spec §5.1 / §5.2 / §5.5）

**顶部 Tab 组件**（新建 `components/layout/mode-tabs.tsx`）：

- 三档 `[🌱 学习] [🔁 复习] [🔥 面试]`，位于中栏顶部（ChatArea 上方）
- 当前 activeMode 高亮，主色随模式（学习 primary 靛蓝 / 复习 review 黄 / 面试 interview 红）
- 点击切 activeMode：调 `PATCH /api/sessions/:id { activeMode }` + `setActiveMode` + 触发对应模式首轮（复用 `handleStartReview`/`handleStartInterview` 的"发消息触发 agent"逻辑；切回 learning 不发消息）

**右栏 tab 退化**（改 `right-sidebar.tsx`）：

- 学习模式：保留 roadmap/code/interactive 三 tab（产物子面板）
- 复习模式：右栏固定 ReviewList（隐藏 tab 栏，或仅显示"复习"单 tab）
- 面试模式：右栏固定 InterviewPanel（同上）
- 移除右栏 review/interview tab 入口（模式切换改由顶部 Tab 负责），但 ReviewList/InterviewPanel 组件保留复用

**三模式主色**（改 `globals.css`）：

- **不动 accent**（teal 被 roadmap-mastered 等复用）；右栏 review/interview 面板的主色从 accent 改为 review/interview 变量

**mode-selector 收口**（改 `mode-selector.tsx` + 类型收窄）：

- 移除 interviewer 项，仅保留 warm/strict
