# 031 — 多模型支持：前端 UI + 设置页

> 状态：⬜ 待开始 | 分类：🔵 功能 | 优先级：P0 | 依赖：030
> 完成日期：—

## 目标

实现前端多模型配置 UI：级联选择器（提供商 → API Key → 模型）、设置页面、会话创建时模型选择、聊天区模型显示。

## 背景

后端已支持多 Provider 配置（迭代 030），前端需要提供配置入口和使用界面。

  - 每个选项显示：模型名 + 智能等级 + 价格
  - 按 tier 分组（旗舰 / 日常 / 性价比 / 轻量）

## 设计系统匹配

- 使用现有暖色设计系统（护眼暗色主题）
- Provider 图标颜色匹配 amber 强调色系
- 卡片/按钮遵循 shadcn/ui 现有组件风格
- 模型 tier 用颜色标签区分：
  - 旗舰 → amber
  - 日常 → stone
  - 性价比 → green
  - 轻量 → gray
