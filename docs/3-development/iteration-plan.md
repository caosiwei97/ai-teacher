## 各迭代详细说明

### 001 项目骨架 + Docker 环境

**目标**：可运行的空项目 + Docker 一键启动

- [ ] Next.js 15 + TypeScript 项目初始化
- [ ] Worker 项目初始化
- [ ] Docker Compose 配置（PostgreSQL + pgvector, Redis, MinIO）
- [ ] Prisma 初始化 + 基础配置
- [ ] npm workspaces 配置（apps/web, apps/worker, packages/*）
- [ ] ESLint + Prettier + TypeScript strict
- [ ] `.env.example` + 环境变量管理
- [ ] 验证：`docker compose up -d` 所有服务正常启动

### 002 数据模型 + Prisma Schema

**目标**：完整的数据库 Schema，与 PRD 数据模型对齐

- [ ] Prisma Schema 定义（User, Source, Session, Roadmap, Node, Message, LearnerProfile）
- [ ] pgvector 扩展（资料向量存储）
- [ ] 迁移脚本
- [ ] 种子数据脚本（测试用）
- [ ] 验证：`npm run db:migrate` 成功，基础 CRUD 可用

### 003 苏格拉底追问 Agent 核心

**目标**：可对话的苏格拉底式教学 Agent（纯 API，无 UI）

- [ ] Tutor Agent system prompt 设计（参考 Sigma Skill）
- [ ] Agent Loop 实现（AI SDK streamText + tool calling）
- [ ] 掌握度评估工具（assessMastery）
- [ ] 知识点追踪工具（recordStrength / recordMisconception）
- [ ] 上下文管理（当前节点 + 已掌握节点 + 学习者画像）
- [ ] 流式响应（SSE）
- [ ] 验证：通过 API 能进行多轮追问教学

### 004 Chat UI + 三栏布局

**目标**：完整的对话界面

- [ ] 三栏布局（左侧栏 + 对话区 + 右侧路线图）
- [ ] 对话气泡组件（支持文本、代码块、评估卡片）
- [ ] 输入框 + 发送
- [ ] SSE 流式接收 + 打字机效果
- [ ] 响应式适配
- [ ] 验证：能与 Agent 进行流式对话

### 005 知识图谱生成 + 可视化

**目标**：自动生成并展示学习路径

- [ ] Roadmap Agent（根据主题/资料生成节点序列）
- [ ] Zod Schema 约束输出（RoadmapOutput）
- [ ] 前端节点列表/路径可视化
- [ ] 节点状态标识（✓ 已掌握 / ● 进行中 / ○ 未开始）
- [ ] 进度条
- [ ] 验证：输入主题 → 生成 5-15 个节点的知识图谱

### 006 诊断摸底

**目标**：新会话开始时评估用户水平

- [ ] 诊断问题生成（3-5 道混合题）
- [ ] 答案评估
- [ ] 起始节点定位
- [ ] 前端诊断 UI（选择题 + 简答题）
- [ ] 验证：开始学习 → 回答诊断问题 → 从正确节点开始

### 007 掌握度评分 + 门控

**目标**：每个节点有明确的掌握度，达标才解锁

- [ ] 每轮对话后 Agent 输出结构化评估（MasteryAssessment Schema）
- [ ] 80% 门控逻辑
- [ ] 节点状态更新（not-started → in-progress → mastered）
- [ ] 前端实时更新节点状态
- [ ] 验证：学习一个节点 → 评分 → 达标后解锁下一节点

### 008 评估卡片渲染

**目标**：节点完成后的总结卡片

- [ ] 评估卡片结构化数据（总结 + 回顾表格 + 核心标签）
- [ ] 前端卡片组件
- [ ] "下一节"按钮
- [ ] 验证：节点掌握后显示评估卡片

### 009 学习资料上传 + RAG

**目标**：用户上传资料，Agent 基于资料教学

- [ ] PDF 解析（pdf-parse）
- [ ] Markdown 解析
- [ ] 文档分块（Chunk）
- [ ] 向量化 + 存储到 PostgreSQL (pgvector)
- [ ] 相似度检索
- [ ] 上传 UI
- [ ] MinIO 文件存储
- [ ] 验证：上传 PDF → 系统解析 → 基于资料内容教学

### 010 会话管理 + 历史记录

**目标**：完整的会话生命周期管理

- [ ] 左侧栏会话列表（学习中 / 已完成）
- [ ] 新建会话
- [ ] 会话恢复（断点续学）
- [ ] 会话删除/归档
- [ ] 验证：关闭浏览器 → 重新打开 → 继续上次学习

### 011 学习者画像 + 跨会话记忆

**目标**：跨会话记住用户信息

- [ ] 学习者画像数据结构（学习偏好、擅长领域、误解模式）
- [ ] 每次会话结束后更新画像
- [ ] 新会话注入画像到 Agent prompt
- [ ] 验证：第一个会话学了 A → 第二个会话 AI 知道你学过 A

### 012 快问 + AI 建议回复

**目标**：辅助功能

- [ ] 快问：选中内容 → 提取上下文 → 调用 LLM
- [ ] AI 建议回复：用户不知道怎么答时提供提示
- [ ] 验证：选中一段文字 → 弹出快问 → 得到上下文相关的回答
