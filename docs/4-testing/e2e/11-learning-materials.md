# ## K. 学习资料

> [← 返回测试矩阵总览](./README.md)

### E2E-K01 PDF 资料上传与解析

**对应 同类苏格拉底教学产品 交互**：支持上传 PDF 资料并基于资料内容教学

#### 前置条件
- 应用已启动

#### 步骤

| # | 操作 | 期望结果 |
|---|------|----------|
| 1 | 在首页找到上传入口 | 有上传按钮或拖拽区域 |
| 2 | 上传一个 PDF 文件 | 文件上传成功，显示文件名和解析状态 |
| 3 | 等待解析完成 | 状态变为 "解析完成" |
| 4 | 基于资料创建学习会话 | AI 基于资料内容生成路线图和教学 |
| 5 | 验证教学内容 | 追问的知识点来自上传的资料 |

#### 断言
- [ ] 上传入口可见
- [ ] PDF 上传和解析成功
- [ ] 路线图基于资料内容生成
- [ ] 教学内容引用资料中的知识点

### E2E-K02 无资料模式学习

**对应 同类苏格拉底教学产品 交互**：不上传资料也能正常使用全部功能

#### 前置条件
- 应用已启动

#### 步骤

| # | 操作 | 期望结果 |
|---|------|----------|
| 1 | 不上传任何资料，直接输入主题 | 正常创建会话 |
| 2 | 完成诊断和路线图生成 | 路线图基于 AI 通用知识生成 |
| 3 | 进行教学对话 | 教学内容准确、有深度 |

#### 断言
- [ ] 不上传资料也能完成全流程
- [ ] 路线图合理且完整
- [ ] 教学内容准确
