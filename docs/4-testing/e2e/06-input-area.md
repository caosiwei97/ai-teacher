# ## F. 输入区功能

### E2E-F01 教学模式切换

**对应 同类苏格拉底教学产品 交互**：输入区 "教学模式" 按钮，支持三种模式切换

#### 前置条件
- E2E-B01 完成

#### 步骤

| # | 操作 | 期望结果 |
|---|------|----------|
| 1 | 验证默认模式 | 教学模式按钮显示当前模式名称 |
| 2 | 点击教学模式按钮 | 弹出下拉菜单，显示 3 个选项 |
| 3 | 验证选项内容 | 温暖私教（"慢慢来，不着急"）、严格教练（"不接受'差不多就行'"）、面试官（"像真实面试一样"） |
| 4 | 验证选项样式 | 每个选项有图标 + 名称 + 描述，当前模式高亮 |
| 5 | 切换到 "严格教练" | 按钮文本更新为 "严格教练" |
| 6 | 继续对话 | AI 的追问风格变严格 |

#### 断言
- [ ] 按钮显示当前模式名称
- [ ] 下拉菜单有 3 个选项
- [ ] 每个选项有图标 + 名称 + 描述
- [ ] 当前模式高亮
- [ ] 切换后按钮文本更新
- [ ] 切换后 AI 风格有明显变化

### E2E-F05 发送按钮状态变化

**对应 同类苏格拉底教学产品 交互**：发送按钮在空输入时 disabled，流式输出时变为 "停止生成"

#### 前置条件
- E2E-B01 完成

#### 步骤

| # | 操作 | 期望结果 |
|---|------|----------|
| 1 | 验证空输入状态 | 输入框为空时，发送按钮 disabled |
| 2 | 输入文字 | 发送按钮变为可点击 |
| 3 | 发送消息 | AI 开始流式响应，按钮变为 "停止生成" |
| 4 | AI 响应完成 | 按钮恢复为 "发送"（disabled，因输入框为空） |

#### 断言
- [ ] 空输入时发送按钮 disabled
- [ ] 有输入时发送按钮可点击
- [ ] 流式输出时按钮变为 "停止生成"
- [ ] 完成后按钮恢复
