## 文档结构

```
docs/
├── 产品/                          ← 产品定义（WHAT）
│   ├── 产品定位.md                 ← 项目一句话、核心场景、用户、约束
│   └── 需求规格.md                 ← 用户故事、功能规格、MVP 范围
├── 设计/                          ← 技术方案（HOW）
│   ├── 技术架构.md                 ← 技术栈、系统架构、数据模型
│   ├── API接口.md                  ← 已实现的接口文档
│   ├── Prompt设计.md               ← 苏格拉底 Prompt + Agent 工具定义
│   ├── 设计系统.md                 ← UI 设计系统规范
│   ├── 错误降级策略.md             ← 错误处理、重试、降级、超时策略
│   ├── 安全设计.md                 ← 沙箱安全、密钥管理、威胁模型
│   ├── 决策记录.md                 ← 关键技术决策及原因（ADR）
│   ├── 架构升级方案.md             ← ⚠️ 已归档 — 迭代 016-022 已实施
│   ├── Agent引擎优化方案.md        ← ⚠️ 已归档 — 迭代 011-012 已实施
│   └── diagrams/                  ← 架构图表（Excalidraw + PNG）
├── 开发/                          ← 迭代进展（WHEN & PROGRESS）
│   ├── 迭代计划.md                 ← 索引：总览表 + E2E 映射 + 详情链接
│   ├── 迭代详情/                   ← 每个迭代的完整详情（按需加载）
│   │   ├── 001-项目骨架.md
│   │   └── ...
│   ├── 开发日志.md                 ← 日志索引 + 里程碑 + 继续点
│   └── 开发日志/                   ← 按年月归档的精简开发日志
│       └── 2026/05/
│           ├── 08.md
│           └── ...
└── 测试/                          ← 测试相关
    ├── 发版测试清单.md              ← 发版前必须通过的测试清单
    └── e2e/                       ← E2E 测试矩阵总览
        └── README.md
```
