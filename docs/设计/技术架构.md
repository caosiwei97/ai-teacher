## 1. 技术栈

| 层 | 技术 | 理由 |
|---|------|------|
| Runtime | Node.js 20 LTS, TypeScript strict | 生态成熟，AI SDK 原生支持 |
| Web | Next.js 15 (App Router), React 19 | SSR + API Routes 一体化 |
| UI | shadcn/ui + Tailwind CSS 4 | 快速开发，暖色设计系统 |
| Agent | AI SDK v4 (streamText + tool calling) | 成熟的 Agent Loop，流式输出 |
| LLM | 智谱 GLM（OpenAI 兼容接口） | 国内可用，性价比高，默认 `glm-4-flash` |
| DB | PostgreSQL 16 + pgvector + Prisma ORM | 结构化数据 + 向量检索（RAG 预留） |
| Cache | Redis 7 | 会话状态 + 对话缓存（预留） |
| 存储 | MinIO (S3 兼容) | 用户上传的文件存储（预留） |
| 容器 | Docker Compose | 一键启动所有中间件 |
| 包管理 | pnpm workspaces | Monorepo 依赖管理 |

## 4. 数据模型

### 4.1 Prisma Schema（实际实现）

```prisma
model User {
  id        String          @id @default(cuid()) @db.Text
  name      String          @db.Text
  sessions  Session[]
  sources   Source[]
  profile   LearnerProfile?
  createdAt DateTime        @default(now())
}

model Source {
  id        String    @id @default(cuid()) @db.Text
  userId    String    @db.Text
  user      User      @relation(fields: [userId], references: [id])
  title     String    @db.Text
  type      String    @db.Text          // pdf | markdown | url
  content   String?   @db.Text          // 解析后的文本
  fileUrl   String?   @db.Text          // MinIO 文件 URL
  checksum  String?   @db.Text          // 文件哈希
  sessions  Session[]
  createdAt DateTime  @default(now())
}

model Session {
  id        String    @id @default(cuid()) @db.Text
  userId    String    @db.Text
  user      User      @relation(fields: [userId], references: [id])
  topic     String    @db.Text
  sourceId  String?   @db.Text
  source    Source?   @relation(fields: [sourceId], references: [id])
  status    String    @default("diagnosing") @db.Text  // diagnosing | active | completed | archived
  roadmap   Roadmap?
  messages  Message[]
  createdAt DateTime  @default(now())
  updatedAt DateTime  @updatedAt
}

model Roadmap {
  id        String  @id @default(cuid()) @db.Text
  sessionId String  @unique @db.Text
  session   Session @relation(fields: [sessionId], references: [id])
  nodes     Node[]
  version   Int     @default(1)
}

model Node {
  id           String   @id @default(cuid()) @db.Text
  roadmapId    String   @db.Text
  roadmap      Roadmap  @relation(fields: [roadmapId], references: [id])
  index        Int      // 排序
  title        String   @db.Text
  description  String   @db.Text
  status       String   @default("not-started") @db.Text  // not-started | in-progress | mastered
  masteryScore Int      @default(0)                         // 0-100
  reviewLog    Json?    // 评估回顾数据
  masteredAt   DateTime?
}

model Message {
  id        String   @id @default(cuid()) @db.Text
  sessionId String   @db.Text
  session   Session  @relation(fields: [sessionId], references: [id])
  role      String   @db.Text    // tutor | learner | system
  type      String   @db.Text    // text | quiz | quiz_response | assessment | system
  content   String   @db.Text
  metadata  Json?    // 结构化数据（工具调用结果、评估卡片等）
  createdAt DateTime @default(now())
}

model LearnerProfile {
  id                   String   @id @default(cuid()) @db.Text
  userId               String   @unique @db.Text
  user                 User     @relation(fields: [userId], references: [id])
  learningStyle        Json?    // 学习偏好
  strengths            Json?    // 擅长领域
  weaknesses           Json?    // 薄弱领域
  misconceptionPatterns Json?   // 常见误解模式
  sessionsSummary      Json?    // 历史学习摘要
  updatedAt            DateTime @updatedAt
}
```

## 5. Agent 架构

### 5.1 Tutor Agent（核心）

```typescript
// 使用 AI SDK streamText + 智谱 provider
const result = await streamText({
  model: openai('glm-4-flash'),  // 智谱 OpenAI 兼容接口
  system: buildTutorSystemPrompt(context),
  messages: [...],
  tools: { assessMastery, recordStrength, recordMisconception, generateAssessment, advanceNode },
});
```

### 5.2 Diagnostic Agent

```typescript
// 使用 AI SDK generateObject + Zod Schema 约束输出
const result = await generateObject({
  model: openai('glm-4-flash'),
  schema: DiagnosticOutputSchema,
  prompt: `...`,
});
```

### 5.3 Roadmap Agent

```typescript
// 使用 AI SDK generateObject + Zod Schema 约束输出
const result = await generateObject({
  model: openai('glm-4-flash'),
  schema: RoadmapOutputSchema,  // 约束 5-15 个节点
  prompt: `...`,
});
```

## 7. Docker Compose

```yaml
# 实际配置：仅中间件，Web/Worker 本地开发运行
services:
  postgres:
    image: pgvector/pgvector:pg16
    ports: ["25432:5432"]
    environment:
      POSTGRES_DB: ai_teacher
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: postgres
    volumes: [./data/postgres:/var/lib/postgresql/data]

  redis:
    image: redis:7-alpine
    ports: ["26379:6379"]
    volumes: [./data/redis:/data]

  minio:
    image: minio/minio
    ports: ["29000:9000", "29001:9001"]
    command: server /data --console-address ":9001"
    environment:
      MINIO_ROOT_USER: minioadmin
      MINIO_ROOT_PASSWORD: minioadmin
    volumes: [./data/minio:/data]
```

## 9. 开发环境

```bash
# 一键启动中间件
docker compose up -d

# 安装依赖 + 初始化数据库
pnpm install
pnpm db:generate
pnpm db:migrate
pnpm --filter @ai-teacher/db seed

# 本地开发（热更新）
pnpm dev:web      # Next.js :38421
pnpm dev:worker   # Worker  :38422

# 构建
pnpm --filter @ai-teacher/web build
pnpm --filter @ai-teacher/worker build

# 测试
npx playwright test
```
