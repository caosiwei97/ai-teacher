## 5. 发版步骤

```bash
# 1. 确认全部通过
pnpm build && pnpm test:e2e

# 2. 更新版本号（如需要）

# 3. 打 tag 并推送
git tag -a v{版本号} -m "Release v{版本号}"
git push origin v{版本号}
```

## 6. E2E 失败处理

1. 判断原因：代码 bug 还是测试过时
2. 代码 bug → 修复 → 重跑
3. 测试过时 → 确认新行为符合预期 → 更新测试 → 重跑
4. **绝不允许**：跳过测试、删除失败测试
