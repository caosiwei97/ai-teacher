```
C. 了解基本概念，但实际项目经验不多
D. 刚接触，对核心概念有了解但没怎么实践
E. 不太清楚
其他（自由输入）
```

```

B. 大致了解机制，但某些方面还模糊
C. 主要从 API 层面使用，不太清楚内部原理
D. 不太清楚原理
其他（自由输入）
```

**Tab 3 - 学习目标**

```
A. 深入理解内部机制

其他（自由输入）
```

### 教学模式数据

| 模式     | 名称     | 描述               | AI 风格特征            |
| -------- | -------- | ------------------ | ---------------------- |
| 温暖私教 | 温暖私教 | 慢慢来，不着急     | 耐心、鼓励、多肯定     |
| 严格教练 | 严格教练 | 不接受"差不多就行" | 直接指出问题、要求精准 |
| 面试官   | 面试官   | 像真实面试一样     | 压力追问、考察深度     |
