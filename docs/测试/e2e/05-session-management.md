# ## E. 会话管理

> [← 返回测试矩阵总览](./README.md)

### E2E-E01 左侧栏三态分组（学习中/新对话/已完成）

**对应 同类苏格拉底教学产品 交互**：左侧栏会话按状态分为三组

#### 前置条件
- 存在多个不同状态的会话

#### 步骤

| # | 操作 | 期望结果 |
|---|------|----------|
| 1 | 验证分组存在 | 左侧栏有 "学习中"、"新对话"、"已完成" 三个分组 |
| 2 | 验证 "学习中" 会话 | 显示进度（如 "1/5"），有图标 + 标题 + 私教课 tag |
| 3 | 验证 "新对话" 会话 | 无进度，显示时间（如 "16 天前"、"刚刚"），有图标 + 标题 + 私教课 tag |
| 4 | 验证 "已完成" 会话 | 显示时间，有图标 + 标题 + 私教课 tag |

#### 断言
- [ ] 三个分组均可见
- [ ] 学习中会话显示进度
- [ ] 新对话显示相对时间
- [ ] 已完成显示相对时间
- [ ] 每个会话卡片有图标 + 标题 + 私教课 tag

### E2E-E03 分组展开与折叠

**对应 同类苏格拉底教学产品 交互**："展开其余 N 个" 按钮展开隐藏会话，变为 "收起" 按钮

#### 前置条件
- 已完成会话数量超过默认显示数

#### 步骤

| # | 操作 | 期望结果 |
|---|------|----------|
| 1 | 验证 "展开其余 N 个" 按钮 | 按钮可见，N 为隐藏的会话数 |
| 2 | 点击展开按钮 | 所有隐藏会话显示，按钮变为 "收起" |
| 3 | 点击 "收起" | 回到折叠状态 |
| 4 | 验证展开后的删除按钮 | 展开的已完成会话卡片上出现删除按钮 |

#### 断言
- [ ] 展开/收起切换正常
- [ ] 展开后显示全部会话
- [ ] 收起后回到默认显示数
- [ ] 展开后删除按钮可见

### E2E-E04 会话切换与恢复

**对应 同类苏格拉底教学产品 交互**：点击不同会话切换到对应的学习内容

#### 前置条件
- 多个会话存在

#### 步骤

| # | 操作 | 期望结果 |
|---|------|----------|
| 1 | 点击一个 "学习中" 的会话 | 页面切换到该会话，对话历史和进度恢复 |
| 2 | 点击一个 "已完成" 的会话 | 页面显示完整对话历史，所有节点为 ✓ |
| 3 | 点击一个 "新对话" 的会话 | 页面显示该会话的对话内容 |

#### 断言
- [ ] 切换后对话历史完整
- [ ] 进度状态正确
- [ ] 右侧栏路线图状态与会话匹配

### E2E-E05 会话删除

**对应 同类苏格拉底教学产品 交互**：展开已完成分组后，会话卡片上出现删除按钮

#### 前置条件
- 已完成会话存在，分组已展开

#### 步骤

| # | 操作 | 期望结果 |
|---|------|----------|
| 1 | 在展开的已完成会话卡片上找到删除按钮 | 删除按钮可见 |
| 2 | 点击删除 | 弹出确认对话框 |
| 3 | 确认删除 | 会话从列表中移除 |
| 4 | 验证列表更新 | 分组计数 -1 |

#### 断言
- [ ] 删除按钮仅在展开状态可见
- [ ] 有确认对话框防误删
- [ ] 删除后会话从列表消失
- [ ] 分组计数正确更新

#### 断言
- [ ] 新会话按钮存在且可点击
- [ ] 点击后回到首页
- [ ] 首页内容完整
