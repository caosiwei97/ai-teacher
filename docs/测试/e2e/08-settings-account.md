# ## H. 设置与账户

> [← 返回测试矩阵总览](./README.md)

### E2E-H01 设置面板打开与信息展示

**对应 同类苏格拉底教学产品 交互**：Banner 设置按钮打开设置面板

#### 前置条件
- 用户已登录

#### 步骤

| # | 操作 | 期望结果 |
|---|------|----------|
| 1 | 点击 Banner 区 "设置" 按钮 | 设置面板打开 |
| 2 | 验证用户信息 | 显示头像 + 姓名 + 邮箱 + "Pro" 标签（如有） |
| 3 | 验证用量信息 | 显示 "本月用量"（如 238/1000），含套餐内学力、永久学力、剩余学力明细 |
| 4 | 验证日期信息 | 显示重置日期 + Pro 过期日期 |
| 5 | 验证功能列表 | 语言选择、暗色模式开关、通知开关、导出学习记录、自定义私教指令、兑换码、退出登录 |

#### 断言
- [ ] 设置面板打开正常
- [ ] 用户信息完整
- [ ] 用量信息有明细
- [ ] 所有功能入口可见
