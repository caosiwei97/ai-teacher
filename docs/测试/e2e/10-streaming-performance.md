# ## J. 流式响应与性能

> [← 返回测试矩阵总览](./README.md)

### E2E-J01 流式文字输出

**对应 同类苏格拉底教学产品 交互**：AI 回答逐字/逐段流式显示

#### 前置条件
- E2E-C01 进行中

#### 步骤

| # | 操作 | 期望结果 |
|---|------|----------|
| 1 | 发送一条消息 | AI 开始流式响应 |
| 2 | 验证流式效果 | 文字逐字/逐段出现，有打字机效果 |
| 3 | 验证代码块 | AI 回复中的代码块正确渲染（语法高亮） |
| 4 | 验证粗体/列表 | Markdown 格式正确渲染 |
| 5 | 等待完成 | 流式结束，消息完整 |

#### 断言
- [ ] AI 响应是流式的（非一次性出现）
- [ ] 代码块语法高亮正确
- [ ] Markdown 格式正常渲染
- [ ] 流式结束后消息完整

### E2E-J02 停止生成

**对应 同类苏格拉底教学产品 交互**：流式输出期间发送按钮变为 "停止生成"

#### 前置条件
- E2E-J01 流式输出进行中

#### 步骤

| # | 操作 | 期望结果 |
|---|------|----------|
| 1 | 验证按钮状态 | 流式期间发送按钮变为 "停止生成" |
| 2 | 点击 "停止生成" | AI 输出立即停止 |
| 3 | 验证已输出内容 | 已输出的文字保留 |
| 4 | 验证按钮恢复 | 按钮恢复为 "发送"（disabled） |
| 5 | 输入新消息并发送 | 可正常继续对话 |

#### 断言
- [ ] 流式期间有 "停止生成" 按钮
- [ ] 点击后立即停止
- [ ] 已输出内容保留
- [ ] 停止后可继续正常对话
