### E2E-G01 路线图 Tab 节点状态与点击

**对应 同类苏格拉底教学产品 交互**：右侧栏路线图 Tab 显示节点列表，不同状态有不同样式

#### 前置条件
- E2E-B01 完成，路线图已生成

#### 步骤

| # | 操作 | 期望结果 |
|---|------|----------|
| 1 | 验证路线图 Tab 为默认选中 | "路线图" Tab 高亮 |
| 2 | 验证节点状态样式 | 当前节点高亮，已完成 ✓，未开始灰色/锁定 |
| 3 | 点击一个已完成节点 | 对话跳转到该节点的评估卡片位置 |
| 4 | 点击一个未开始（锁定）节点 | 无法跳转或提示需完成前置节点 |

#### 断言
- [ ] 节点状态样式正确（✓ / 高亮 / 灰色）
- [ ] 已完成节点可点击跳转
- [ ] 锁定节点不可操作

### E2E-G02 学习笔记 Tab 内容渲染

**对应 同类苏格拉底教学产品 交互**：学习笔记 Tab 显示每个节点的笔记内容，不同状态不同展示

#### 前置条件
- E2E-D02 完成，至少一个节点已掌握

#### 步骤

| # | 操作 | 期望结果 |
|---|------|----------|
| 1 | 切换到 "学习笔记" Tab | Tab 切换成功 |
| 2 | 验证已完成节点笔记 | ✓ 标记 + AI 自动生成的总结 + 掌握度评分（如 "85"） |
| 3 | 验证进行中节点 | 显示 "点击添加笔记" |
| 4 | 验证锁定节点 | 锁图标 + "学完前置节点后解锁" |

#### 断言
- [ ] 已完成节点有 ✓ + 总结 + 分数
- [ ] 进行中节点可添加笔记
- [ ] 锁定节点显示锁定提示
